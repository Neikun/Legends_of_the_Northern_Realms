Legend of Grimrock Modifications
By germanny

Updated december 2, 2012 to v1.1

New prop 'Grimrock Bed'
Model: 832 Vertices, 1001 faces.


Installation:---------------------------------------------------------------------------
Copy the folder 'grim_bed' with its contents into your dungeon´s folder 'mod_assets'.

Add following line to your 'init.lua':

import "mod_assets/grim_bed/grim_bed.lua"


Notes:------------------------------------------------------------------------------------
V1.1 = Updated normal map, has to be in 32bit to work correct.
V1.1 = Added second diffuse map with brown sheet, make it default

Change the diffuse texture to 'grim_bed_dif1.tga' for grey sheet!

! Textures may way too big for props.

For the Lua´s: Only example scripts, feel free to tune!
